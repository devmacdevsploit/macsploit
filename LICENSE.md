## 🔧 Common Issues & Fixes

**❌ macOS says "MacSploit cannot be opened because it is from an unidentified developer"**
> This is Gatekeeper blocking the app because MacSploit is not distributed through the App Store. Go to **System Settings → Privacy & Security**, scroll down to the Security section, and click **"Open Anyway"**. You only need to do this once.

**❌ Auto-attach doesn't trigger when Roblox launches**
> Ensure Roblox is fully loaded into a game session (not just the home screen) before auto-attach triggers. If it still doesn't attach, use the Manual Inject option in the Inject tab. Also check that MacSploit has the required accessibility permissions under **System Settings → Privacy & Security → Accessibility**.

**❌ Script executes but produces no result in-game**
> The script may rely on Windows-specific API calls from a Windows-only executor. Check the script source for any Windows executor-specific functions and refer to the MacSploit API documentation in the Help tab for macOS equivalents.

**❌ MacSploit crashes immediately on launch**
> Confirm you are running macOS 13 Ventura or later. If you are on a supported version, delete the MacSploit preferences file at `~/Library/Application Support/MacSploit/` and relaunch — a corrupt preferences file is the most common crash cause.

**❌ Injection fails with "Process access denied" error**
> MacSploit needs Accessibility permissions to inject into Roblox. Go to **System Settings → Privacy & Security → Accessibility** and ensure MacSploit is checked. If it's already listed, remove it and re-add it by dragging the app back in.

**❌ Script Hub shows "Failed to load" error**
> This is a network connectivity issue. Confirm your Mac's internet connection is active and that your firewall or Little Snitch (if installed) is not blocking MacSploit's outbound connections. Whitelist MacSploit in your firewall settings.

**❌ GUI appears blurry on Retina display**
> This occurs when macOS overrides the app's display scaling. Right-click MacSploit in Finder, select **Get Info**, and uncheck **"Open in Low Resolution"** if it is checked. Relaunch the app.
